Add a Reference Image

Reference images are added using the current viewport orientation, so line up your view first.

  1. Choose the axis that best matches the reference image, such as front, side, or top.
  2. Lock the camera/view to that axis before adding the image.
  3. Use Add > Image > Reference.
  4. Select the image file you want to use.

The reference image will be aligned with the current camera view, which is why it is important to get the view locked to the right axis first.
