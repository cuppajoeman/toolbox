Basic Annotation Controls

  • Press D and left-click to draw annotations with the mouse.
  • Press D and right-click to erase annotations.
  • All annotations can be viewed and managed in the View > Annotate section of the N panel. This panel allows users to erase entire layers in a single action.

Annotation Layers

The annotation tool supports multiple layers. Layers enable users to:

  • Organize different annotations by color or purpose.
  • Use onion skinning to see previous layers with distinct coloring.
  • Manage complex annotation sets without visual clutter.

Stabilizing and Smoothing Strokes

While drawing, strokes can be stabilized to create smoother lines. To activate this feature:

  1. Press D to start drawing.
  2. While holding D, press Shift to enable the Stabilize Stroke feature.

Placement Options

Annotations can be placed in three different modes:

  • 3D Cursor: Annotations are drawn in 3D space relative to the cursor position.
  • View: Annotations are drawn directly on the screen viewport.
  • Surface: Annotations are drawn on the surface of objects or characters.

These options are accessible through the Tool menu (T) under the annotation tool section.

Bonus Tips

For efficient erasing, zooming out slightly before using the erase function increases the relative size of the eraser, allowing for faster removal of annotations.
